# koydo-tpat

TPAT exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM081 |
| Name | TPAT |
| Country | Thailand |
| Category | Admissions |
| Tier | Tier 5 |
| Wave | Wave 7 |
